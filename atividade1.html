    Ensinando como instalar a IDE VS Code(Visual Studio Code) e a como utilizar a IDE, baixando plugins facilitadores para o desenvolvimento dos códigos, funcionamentos básicos de como criar uma pasta para trabalharmos, como criar um arquivo HTML e estrutura básica do html:5

  • Aula 3

    • Voltar

    Mostrando a funcionalidade de inspeção de elementos em websites, manipulação dos códigos de websites para estudo e entendimento de como as páginas funcionam, alteração prática de elementos de páginas.

  • Aula 4

    • Voltar

    Construção da estrutura básica do HTML, mostrando a funcionalidade das TAGS: html, head, body, title e também funcionamento do plugin Live Server.

    • Voltar

    • Voltar

    Aprofundando sobre atributos de TAGS, introduzindo os atributos: type(text, number, color), id, style, class, src e width.

  • Aula 7

    • Voltar

    Implementando texto nas páginas HTML, utilizando na prática as TAGS: h(1,2,3,4,5,6), p, blockquote, u, strong, i, mark, sup.

  • Aula 8

    • Voltar

    Introdução das TAGS ul, ol e li, como funcionam as listas ordenadas(ol), listas não ordenadas(ul) e itens que estão nas listas(li)
